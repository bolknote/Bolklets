Host page content

This is the development host page. It loads each bolklets source file individually from js/, which is handy while editing. For production, run make bundle and embed the resulting single bolklets.js instead — see dist/index.html for that setup.

The lawn regenerates on every reload, so no two visits look exactly alike. The scripts are loaded with defer so they never block the main content from rendering.
